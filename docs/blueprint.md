# **App Name**: PEC.AI

## Core Features:

- Image Capture: Allows users to capture images using their device's camera.
- Object Identification: Uses AI to identify the object in the captured image. The tool will try its best to understand objects even if they aren't in the center of the image, occluded or not perfectly aligned.
- Background Removal: Automatically removes the background from the identified object in the image.
- Card Generation: Generates a visual card with the image, object name, and category.
- Card Library: Saves the generated cards in a user-specific library for future use.
- Phrase Assembly: Enables users to create phrases by dragging and arranging cards.
- Spoken Phrase: Speak assembled phrase using built-in text to speech capabilities

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke a sense of calm and communication, referencing the app's use for communication support.
- Background color: Very light blue (#F0F8FF), nearly white, to provide a clean and unobtrusive backdrop, helping focus on the cards.
- Accent color: Muted teal (#77B5B0) to complement the blue, adding a touch of sophistication and indicating interactive elements.
- Body and headline font: 'PT Sans' for clear, readable text suitable for all users.
- Use simple, intuitive icons for navigation and actions.
- Use a clean and spacious layout with easily identifiable buttons and draggable elements.
- Subtle animations to indicate drag-and-drop actions and successful card saves.
